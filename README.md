# Anviz

.NET library to access and control Anviz devices.

Features
--------

 - Device Statistics
 - Download All Records
 - Get All Users
 - Get TCP/IP Parameters
 - Get Device SN
 - Get Device Type Code
 - Clear new records sign 


Tested devices
--------

 - VF Series
 - W1 Pro
